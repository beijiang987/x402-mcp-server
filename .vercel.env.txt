# ============================================
# 添加说明
# ============================================

如何在 Vercel 中添加：
1. 访问 https://vercel.com/dashboard
2. 选择 x402-mcp-server 项目
3. 点击 Settings → Environment Variables
4. 对于每个变量：
   - Key: 变量名（例如 X402_PAYMENT_ADDRESS_BASE）
   - Value: 变量值（复制上面的值）
   - Environment: 勾选 Production, Preview, Development
   - 点击 Add

需要添加的 3 个必需变量：
✅ X402_PAYMENT_ADDRESS_BASE
✅ X402_PAYMENT_ADDRESS_ETH
✅ NODE_ENV
