API 文档

简单易用的 MCP 接口

📊 get_token_price

获取实时代币价格

const price = await client.callTool({
  name: 'get_token_price',
  arguments: {
    token_address: '0xC02a...',
    chain: 'ethereum'
  }
});

🌐 get_multichain_price

跨链价格聚合

const prices = await client.callTool({
  name: 'get_multichain_price',
  arguments: {
    token_symbol: 'USDC',
    chains: ['ethereum', 'base', 'polygon']
  }
});

🔒 scan_contract_safety

合约安全扫描

const safety = await client.callTool({
  name: 'scan_contract_safety',
  arguments: {
    contract_address: '0xC02a...',
    chain: 'ethereum'
  }
});
