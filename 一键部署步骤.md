## 现在只需要做 3 件事

---

### 第 1 步：打开部署页面

**直接点击这个链接**（在浏览器中打开）：
```
https://vercel.com/import/git
```

然后：
1. 找到 `beijiang987/x402-mcp-server` 仓库
2. 点击 "Import" 按钮

---

### 第 2 步：添加环境变量

在 "Configure Project" 页面，找到 "Environment Variables" 部分。

**复制下面的内容，一个一个粘贴**：

#### 变量 1
```
名称：X402_PAYMENT_ADDRESS_BASE
值：0xa893994dbe2ea7dd7e48410638d6a1b1b663b6a3
```

#### 变量 2
```
名称：X402_PAYMENT_ADDRESS_ETH
值：0xa893994dbe2ea7dd7e48410638d6a1b1b663b6a3
```

#### 变量 3
```
名称：NODE_ENV
值：production
```

**⚠️ 重要**：每个变量都要勾选三个环境：
- ✅ Production
- ✅ Preview
- ✅ Development

---

### 第 3 步：点击 Deploy

点击页面底部的蓝色 **"Deploy"** 按钮。

等待 2-3 分钟，部署就会完成！

## 如果您已经创建了项目

如果您之前已经创建了 x402-mcp-server 项目，那么：

### 方法 A：通过项目页面部署

1. 访问：`https://vercel.com/dashboard`
2. 点击 `x402-mcp-server` 项目
3. 点击顶部的 **"Deployments"** 标签
4. 点击右上角的 **"Deploy"** 按钮
5. 选择 `main` 分支
6. 点击 **"Deploy"**

### 方法 B：删除重新导入（更简单）

1. 访问：`https://vercel.com/dashboard`
2. 点击 `x402-mcp-server` 项目
3. 点击顶部的 **"Settings"** 标签
4. 滚动到最底部，点击 **"Delete Project"**
5. 确认删除
6. 然后按照上面的"第 1 步"重新导入

**推荐使用方法 B**，因为可以确保环境变量设置正确。
